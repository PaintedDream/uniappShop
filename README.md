# uniappShop
简介
=====

此项目是一个用uniapp框架编写的一个移动端商城，
全项目使用官方推荐的rpx作为单位布局，用了少量的uniui库中的组件,
目前完成了，首页，分类，商品详情，
评价，还差购物车未完成，是一个uniapp练笔
